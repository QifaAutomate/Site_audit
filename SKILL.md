# Site Text Audit

## Назначение

Проверяет сайт QIFA на ошибки текстового контента.

Сайт фиксированный:

https://www.qifa.ru

Проверяет:
- орфографические ошибки
- грамматические ошибки
- пунктуационные ошибки
- логические ошибки
- стилистические ошибки

Результат сохраняется в Excel-файл.

---

## Когда использовать

Используй этот skill, если пользователь просит:

- проверить сайт на ошибки
- сделать аудит сайта
- проверить орфографию сайта
- проверить грамматику сайта
- найти ошибки на сайте
- проверить текстовый контент сайта
- проверить конкретный модуль сайта
- сделать SEO-аудит сайта
- проверить SEO сайта

---

## Модули

Доступные модули:

- `home` — главная страница
- `full_site` — полный обход сайта
- `catalog` — каталог
- `product` — карточки товаров
- `help` — помощь, FAQ, инструкции
- `news` — новости
- `blog` — статьи / блог
- `company` — о компании / платформа
- `contacts` — контакты
- `legal` — юридические страницы
- `marketing` — маркетинговые и лендинговые страницы
- `search` — поиск
- `user` — личный кабинет, если доступен
